Classes in Ruby

An explanation and demonstration

November 7, 2015

This past week at Dev Bootcamp, we focused on learning about classes in Ruby, and I would liek to share some of what knowledge I have gleaned.

A class is type of object (as are most things in Ruby), but this object serves as a container for other objects, attributes, and methods. It is a way to organize and define a set of objects that share the same properties and pre-defined methods that can be used on them.

Ruby has a number of pre-defined classes for object types such as Integer, String, and Array. Using these pre-defined classes, you can summon a new instance of that class like so:

Summoning a new object

list = Array.new
list.is_a? Array
=> true

You may also define a new class from scratch, as in the example below:

Class Example

Creating a new class
class Wand

def initialize(charges)
unless (charges.is_a? Integer) && (charges >= 0)
raise ArgumentError.new("Must specify a positive integer of charges for a new wand.")
end
@charges = charges
end

def charges_left
@charges
end

def magic_missle
if @charges <= 0
puts "Out of juice"
else
puts "Blasts enemy with magical force! #{(rand(3)+2)} damage!"
@charges -= 1
end

I will take you through the above code step-by step:

  • The new class is declared
  • The initialization step is set to accept one argument, and if that argument does not fit specifications, it raises an Argument Error.
  • If the argument passes spec, it sets the Instance Variable @charges. The instance variable differs from a local variable in that it is accessible across all instances of that class object, and may be used amongst the methods in its class, unlike a local variable, which is only usable in the space where it is defined.
  • Then, I define just a method for the class called #charges_left, which will return the instance variable @charges.
  • Finally, we give the class some power by defining another method, #magic_missle, which returns one of two strings, depending on the value of @charges.

Now, we will run the code through the steps to demonstrate:

Wielding the wand

sparkle_blaster = Wand.new(50)
sparkle_blaster.charges
=> 50
sparkle_blaster.magic_missle
=> "Blasts enemy with magical force! 3 damage!"
sparkle_blaster.magic_missle
=> "Blasts enemy with magical force! 5 damage!"
sparkle_blaster.charges
=> 48

This is how we make magic items in Ruby. Now you're a wizard too!
